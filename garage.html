Online Garage Management System

Overview

This web application helps manage customers, vehicles, bookings, and service records for a garage.

Technologies Used

  • PHP & MySQL (Backend)
  • HTML, CSS, JavaScript (Frontend)
  • XAMPP Local Server

Features

  • Add / Edit / Delete Customers & Vehicles
  • Manage service records
  • Staff & Task assignment
  • Booking and appointment system
