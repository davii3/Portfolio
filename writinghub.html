Henry Onyango Writing Hub

Overview

Henry Onyango Writing Hub is a personal writing and content creation platform showcasing creative, academic, and technical writing samples. It provides clients and readers with access to well-organized content categories, article previews, and downloadable works.

Technologies Used

  • HTML, CSS, and PHP for page structure and dynamic content.
  • MySQL database for storing articles and client requests.
  • JavaScript for interactive reading sections.

Features

  • Article categories (Content Writing, Technical Writing, Editing, Copywriting)
  • Portfolio gallery for samples and downloadable documents
  • Contact form for writing service requests
  • Responsive and SEO-friendly design
